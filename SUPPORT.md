# Support

Thanks for using the Trading Card API SDK for PHP. Here is where to get help.

## Questions and discussion

- **General questions / ideas:** start a thread in
  [GitHub Discussions](https://github.com/cardtechie/tradingcardapi-sdk-php/discussions).
- **API reference:** see the
  [Trading Card API documentation](https://docs.tradingcardapi.com).

## Bugs and feature requests

- **Bugs:** open a
  [bug report](https://github.com/cardtechie/tradingcardapi-sdk-php/issues/new?template=bug_report.yml).
- **Feature requests:** open a
  [feature request](https://github.com/cardtechie/tradingcardapi-sdk-php/issues/new?template=feature_request.yml).

Before filing, please search existing
[issues](https://github.com/cardtechie/tradingcardapi-sdk-php/issues) and
[discussions](https://github.com/cardtechie/tradingcardapi-sdk-php/discussions)
in case your topic is already covered.

## Documentation in this repository

- [README](README.md) — installation, configuration, and usage.
- [Error Handling Guide](docs/ERROR-HANDLING.md)
- [Response Validation](docs/VALIDATION.md)
- [Version Management](docs/VERSION-MANAGEMENT.md)

## Security

Please do **not** report security vulnerabilities through public issues. See the
[Security Policy](https://github.com/cardtechie/tradingcardapi-sdk-php/security/policy)
for responsible-disclosure instructions.
